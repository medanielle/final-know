## Which of the following symbols are permitted within an identifier name?

KSAs: 14, 24, 28

## Answer
| A | B | ***C*** | D |
| :--- | :--- | :--- | :--- |
| $ | @ | ***_*** | . |


Feedback:

- The underscore char _ is the only valid char listed among these special chars that are allowed within an identifier name.
