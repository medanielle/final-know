## Instructions like MOV or ADD is called a(n) __________ in Assembly.

KSAs: 578, 580, 1166

## Answer
| A | B | ***C*** | D |
| :--- | :--- | :--- | :--- |
| Command | Interrupt | ***Opcode*** | Instruction |


Feedback:

- Opcode is short for operational code and is the terminology used for commands like MOV and ADD.
