## In C programming, what command releases memory that was created on the heap?

KSAs: 295, 296

## Answer
| A | ***B*** | C | D |
| :--- | :--- | :--- | :--- |
| memfree | ***free*** | dealloc | delete |


Feedback:

- The free function is what is used to release any memory allocated with malloc.
